The Basics

HTML

HTML is a semantic markup language for adding context to documents. HTML documents attempt to provide meaning to information on a page (and have nothing to do with style).

CSS

CSS is a language for creating style rules. It allows associating particular pieces of markup with particular rules for color, emphasis, and positioning.

Glossary

Pre-processor
"Pre-processors" are languages that compile into other languages. So, for example, you can write Sass, then it is processed into CSS. Pre-processors make it possible to write less code and do less math.
Semantic
Relating to meaning; semantic code focuses on applying tags, classes, and ids that reflect the meaning of the content. For example, in HTML, a heading should be put into a h tag, such as h1, h2, h3.
Package Manager
Modern front-end coders often work with open source "packages", such as javascript libraries, that add a specific piece of functionality or style needed for the project. Package managers speed up installation of libraries and other "plugin" functionality.
Debug
Figure out what is wrong with your code!
Programming Language
A language that implements standardized constructs to allow communicating behavioral instructions to a computer.
Version Control
Version control is the term for how coders keep track of changes over time. It helps you protect and share your projects.
GUI or Graphical User Interface
GUI is a term used for software that provides a graphical user interface, so that users can follow visual cues rather than writing machine code. For example, Git can be used from the console or with GUI software, such as Tower or SourceTree.
